# Example: open-source project watchtower

This public-safe example tracks public open-source release and documentation pages. It avoids private workspace data, internal company material, confidential organization names, and outbound actions.

## Use case

Collect public release/change signals from open-source browser automation projects. Produce source URLs, screenshots, direct evidence, inference, confidence, recommended action, review questions, Markdown, and JSON.

## Start

```bash
forgewright bootstrap
forgewright examples
```

Then ask your AI:

```text
Use the open-source-project-watchtower example. Show the source registry, explain the evidence contract, then walk me through a first run, local SkillOpt-style tuning, and dashboard refresh.
```
