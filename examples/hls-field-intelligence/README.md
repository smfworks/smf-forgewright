# Example: public HLS field intelligence

This example demonstrates a source-seeded public intelligence collector for healthcare and life sciences field teams.

## Use case

Collect public account, Microsoft, competitor, and market signals. Produce source URLs, screenshots, direct evidence, inference, confidence, seller motion, discovery questions, Markdown, and JSON.

## Start

```bash
forgewright init-use-case
```

Use title: `Public HLS field intelligence`.

Then replace the example source registry entries with known public official and credible sources.
