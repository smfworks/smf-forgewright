# Hermes guide for SMF Forgewright

Hermes can use Forgewright as a disciplined browser-workflow and self-tuning package.

## Suggested Hermes prompt

```text
Read AGENTS.md and HERMES.md. Install or verify SMF Forgewright, present sample use cases, help me configure one, then run and tune it using the local SkillOpt-style workflow.
```

## Hermes responsibilities

- Keep user intent and safety boundaries explicit.
- Produce reusable artifacts rather than one-off browser state.
- Favor source registries over broad search for scheduled work.
- Keep dashboards fed by structured JSON.
- Create promotion packets for human review before changing live playbooks.

## Good first use cases

- Public customer/news intelligence.
- Product release monitoring.
- Competitive public-source watchtower.
- Regulatory or policy page monitoring.
- Public documentation change tracking.
